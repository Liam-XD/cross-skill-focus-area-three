# Playwright End-to-End Testing Project

This project is set up for end-to-end testing using [Playwright](https://playwright.dev/) with TypeScript support.

## Getting Started

### 1. Install dependencies

```
npm install
npx playwright install
```

### 2. Run tests

```
npx playwright test
```

### 3. Open Playwright Test Runner UI

```
npx playwright test --ui
```

## Project Structure

- `tests/` — Example Playwright test files
- `playwright.config.ts` — Playwright configuration

## Writing Tests

- Add new test files in the `tests/` directory.
- See [Playwright documentation](https://playwright.dev/docs/intro) for more details.

## Useful Commands

- `npx playwright codegen` — Record user actions and generate tests
- `npx playwright test --debug` — Run tests with debugging
